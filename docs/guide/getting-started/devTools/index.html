We assume that you got the Vue dev tools installed for your browser. The extension is available for Chrome, Firefox, Edge and as a standalone app.

Development setupâ€‹

Furthermore you should have Shopware setup. To make use of the Extension API plugin for the Vue dev tools start the watcher of the administration:

$ composer run watch:admin

Vue dev tool pluginâ€‹

Once you logged into your administration, open up the development tools of your browser and choose the Vue tab. Inside the Vue tab choose the Shopware Extension API plugin.

Usageâ€‹

The plugin will show you all extension points for the current page you visit. Once you select an extension point it will highlight the corresponding area in the viewport and give detailed information how to extend the highlighted property.
